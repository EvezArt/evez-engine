# Client Onboarding Checklist (EVEZ AI Automation)

## Day 0 - Discovery Call (15 min)
- [ ] Ask about their biggest time waster
- [ ] Identify 3-5 questions the agent should answer
- [ ] Confirm their preferred channel (WhatsApp / Telegram / SMS)
- [ ] Get their calendar link
- [ ] Send contract + first month invoice

## Day 1 - Setup (30-60 min)
- [ ] Create dedicated OpenClaw agent instance
- [ ] Connect to their messaging channel
- [ ] Load knowledge base (FAQ, services, hours, policies)
- [ ] Set escalation rules (when to text human)
- [ ] Test with 5 sample conversations

## Day 2 - Handover
- [ ] 20-minute walkthrough with client
- [ ] Show them how to view logs and daily summary
- [ ] Give them direct line to EvezVearl Bot for support
- [ ] Set first weekly review date

## Ongoing
- [ ] Weekly performance report (auto-generated)
- [ ] Monthly optimization call (15 min)
- [ ] Quarterly upsell review

**Goal:** From signed contract to live agent in under 48 hours.
